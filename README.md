# React Essentials

A foundational React project covering core concepts:
- **Components**: Building blocks of the UI.
- **JSX**: JavaScript XML for defining component structure.
- **Props**: Passing data to components.
- **State**: Managing dynamic data and UI updates.

## Features
- Interactive examples of core React concepts.
- Styled using vanilla CSS.
- Built with Vite for a fast development experience.

## Getting Started
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
